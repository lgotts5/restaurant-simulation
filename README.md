# Mimi's Place

A GUI restaurant simulation built with Java Swing. Move a waiter around the floor, take orders from tables, and browse the menu — with a surprise feature.

**Author:** Lauren Gottlieb  
**Date:** December 2023

---

## How to Run

### Requirements
- Java 11 or higher — check with `java -version` in your terminal
- If you don't have Java: [download it here](https://www.java.com/en/download/)

### Option 1 — Command line

```bash
git clone https://github.com/lgotts5/restaurant-simulation.git
cd restaurant-simulation
javac *.java
java Restaurant
```

### Option 2 — BlueJ

1. Download and install [BlueJ](https://www.bluej.org)
2. Open BlueJ → **Project** → **Open Project** → select the `restaurant-simulation` folder
3. Right-click the `Restaurant` class → **void main(String[] args)** → **OK**

---

## Controls

- **Arrow keys** — move the waiter around the floor
- **Take Order button** — appears when you're near a table
- **Menu button** (top left) — view the menu (has a surprise feature)

Expand the window to fill your screen for the best experience.
