PROJECT TITLE: Mimi's place
PURPOSE OF PROJECT: GUI restaurant that utilizes a map for the tables.
VERSION or DATE: 12/17/2023
HOW TO USE THIS PROJECT: Run the main method of the restaurant class and expand the window of the program to fit your whole screen. Use the arrow keys to move the waiter around and when you get close to a table you can take orders by pressing the "take order" button that appears. Also, you can view the menu by pressing the menu button in the top left corner which has a surprise feature in it.
